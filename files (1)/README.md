# Assets

This folder is a placeholder for avatar sprite assets used by the frontend's
avatar evolution feature (see PRD.md's Habit / Engagement Design section).

No final art is included here yet — this project doesn't have real sprite files
to bundle. When your team has actual pixel-art stages ready, place them here as:

```
assets/avatar_stage_1.png
assets/avatar_stage_2.png
assets/avatar_stage_3.png
```

and update the frontend's avatar-state logic to reference these paths instead of
placeholder/dummy sprites.

Do not generate or bundle placeholder "AI-generated" character art here without
the team explicitly deciding on an art direction first — inconsistent placeholder
art tends to get left in by accident under hackathon time pressure and ends up in
the final demo.
